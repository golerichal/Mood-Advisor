
# 🤗 Mood Advisor

Mood Advisor is a simple mood-based advice system created using Python and Streamlit.

The user can select their current mood and the system provides a simple message and advice based on the selected mood.

## ✨ Features

- Select your current mood
- Supports different moods like Happy, Sad, Angry, Stressed and Tired
- Displays mood-based messages
- Provides simple advice
- Easy and simple user interface

## 🛠️ Technologies Used

- Python
- Streamlit

## 🚀 How to Run

1. Install Streamlit

pip install streamlit

2. Run the project

streamlit run mood.py

## 💡 What I Learned

Through this project, I learned how to use Streamlit to create a simple web application using Python. I also understood how to use selectbox, conditional statements and different Streamlit message functions.
